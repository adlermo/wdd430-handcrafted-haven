# Handcrafted Haven

Handcrafted Haven is a web application created for the WDD430 group project.
It serves as a small marketplace where artisans can create profiles, list handcrafted items, and connect with users who appreciate handmade products.

### Features

- Seller profiles with bio and product gallery
- Product listings with images, description, and price
- Product reviews and ratings
- Browsing and filtering of items
- Responsive and accessible design
- Built with Next.js, Node.js, and a database (TBD)

### Tech Stack

- **Frontend**: Next.js
- **Backend**: Node.js (API Routes)
- **Database**: To be chosen by the group
- **Deployment**: Vercel
- **Management**: GitHub Projects

### Getting Started

```
git clone https://github.com/YOUR-USERNAME/handcrafted-haven.git
cd handcrafted-haven
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.
